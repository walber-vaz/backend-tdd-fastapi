# Projetos backend usando Fastapi com TDD

## O projeto

O projeto consiste em uma API para gerenciamento de usuários e seus endereços.

## Tecnologias

- Python 3.11.5
- Fastapi
- Pydantic
- SQLAlchemy
- Postgres
- Docker
- Docker-compose
- Poetry
- Flake8
- Black
- Pytest
- Pytest-cov
